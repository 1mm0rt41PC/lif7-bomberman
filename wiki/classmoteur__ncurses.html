Renvoie la couleur d'un joueur.

Paramètres:
[in] joueur Numéro du joueur de 1 à ...
Renvoie:
Renvoie la couleur à utiliser ( couleur directement utilisable dans wattron() )

unsigned int moteur_ncurses::menu ( const char  titre[],
const char *  choix[],
unsigned int  nb_choix 
)

Créé une sous fenêtre centré.

Paramètres:
[in] titre Titre de la sous fenêtre
[in] choix Liste des choix a mettre dans la sous fenêtre. ( Tableau de chaine de caractères )
[in] nb_choix Nombre d'élement dans la liste des choix
Renvoie:
Le choix selectionné (sous forme d'un nombre: le ième element de la liste a été selectionné)

La sous fenêtre propose de naviguer parmit une liste ( choix )
Puis renvoie le choix (sous forme d'un nombre: ième element de la liste)

 // Exemple
 const char* choix[] = {
    "Choix 1",
    "Choix 2",
    "Choix 3",
    "Choix 4",
    "Choix 5"
 };
 unsigned int choix_selectionner = menu( "Liste de mes choix", choix, 5 );
