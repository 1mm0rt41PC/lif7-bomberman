Compilation du projet

Le projet a été conçu pour compiler avec GCC et make (gnu).
En fonction de ce que vous voulez obtenir, les dépendances varies.
Listes des dépendances et des commandes make approriées

  • NCurses
    La lib NCurses ou PDCurses sous Windows 
    make nurcses
  • SDL
    Les Lib :
    SDLmain, SDL, SDL_ttf, SDL_image 
    make sdl
  • SFML
    make sfml

Pour debugguer le programme:

  • memcheck: Permet de détecter les fuites mémoires ( le rapport est dans ./bin/allocations.txt )
  • debug: Permet l'utilisation de gdb ( level 2 ) 
    make memcheck debug
// NOTE: L'odre des commandes importe peu


    Pour plus d'informations sur les options de compilations, cf : 
    make help
