bool map::getModifications ( s_Coordonnees &  pos  ) 

Retourne les positions X,Y des case modifiées depuis le dernier cycle.

Paramètres:
[out] pos La variables où sera stockée les données de retour
Renvoie:
True s'il y a eu des modifications

Utilisation: 

 map myMap;
 s_Coordonnees pos={0,0};
 // Charger une map ici + traitement divers
 // ...
 while( myMap.getModifications(pos) )
 {
    // pos contient les coodonnées d'un block modifié
    printf("Le block a l'adresse: X=%u, Y=%u a été modifié.\n", pos.x, pos.y);
 }
 // Fin des block modifié
Note:
A chaque appel de la fonction l'element qui est envoyé a pos est enlevé de la liste des modification (First In Last Out)
